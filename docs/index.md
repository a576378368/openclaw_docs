# OpenClaw Architecture Book

## 深入理解 OpenClaw 系统架构

开源的多通道 AI 网关，用于将聊天平台与 AI Agent 连接起来。

[开始阅读](zh/architecture-book/part-1-foundations/01-introduction.md){ .md-button .md-button--primary }

## 本书特点

- **九大章节** - 涵盖从基础概念到高级功能的完整知识体系
- **深入源码** - 结合实际代码解析架构设计和实现细节
- **开发指南** - 包含测试、调试和部署的最佳实践

| 章节 | 篇数 | 内容 |
|------|------|------|

| 核心模块 | 8 篇 | 网关、Agent、会话、内存、工具、MCP、Flow、任务 |
| 插件系统 | 6 篇 | 架构、SDK、契约、运行时、开发指南 |
